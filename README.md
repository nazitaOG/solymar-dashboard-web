# Solymar Dashboard – Frontend

## Requisitos

- Node.js >= 18
- pnpm (recomendado)

---

## Tecnologías

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Zustand
- Zod

---

## Instalación

```bash
pnpm install
```

---

## Correr el proyecto en desarrollo

```bash
pnpm dev
```

Luego abrir:

```
http://localhost:5173
```

---

## Build de producción

```bash
pnpm build
```

---

## Preview del build

```bash
pnpm preview
```
